Question.1. Whats the difference between local and session storage?

Local Storgage: Local Storage is to store data after the browser is closed. It means that data stored in local storage does not expire. .

Session Storage: The session Storage object stores data only for a sessin. It means that the data stored inthe sssion Storage will be deleted when browser is closed.
